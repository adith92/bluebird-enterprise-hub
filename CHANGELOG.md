# Changelog 📝

All notable changes to this project will be documented in this file.

This project follows **Semantic Versioning** (SemVer): `MAJOR.MINOR.PATCH`.

## 0.1.0 - 2026-05-07

- 🎛️ Bluebird Hub web app (GM Dashboard, Sales, Operations, Finance, Clients)
- 🔌 Express API server with PostgreSQL + Drizzle ORM + Zod validation
- 🔐 Session-based auth with username/password
- 🌱 Demo data + demo user seeding

## 0.1.1 - 2026-05-07

- 🟦 Added “Continue with Google” login option (Google ID token verification on API)
- 👑 Seeded `admin/admin` demo account (configurable via `DEMO_ADMIN_*`)
- 📚 Added root README (full emoji) + contributor prompt guide + env examples
